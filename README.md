# Linear-regression-dataset-analysis

## Overview
This project analyzes the performance of the Linear Regression algorithm on multiple datasets.

## Objective
The objective of this project is to apply Linear Regression on different datasets and determine which dataset fits the model best.

## Methodology
Linear Regression is applied separately to multiple datasets and evaluated using performance metrics such as R² score.

## Tools and Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Conclusion
The results help identify which dataset provides the best fit for the Linear Regression model based on evaluation metrics.
